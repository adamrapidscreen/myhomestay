Design Spec
1. Mood

Clean, welcoming, local, modern, relaxed

2. Audience feel

Visitors should feel like they are browsing real local homestays from real owners in a simple, trustworthy, and friendly way.

3. Colors

Primary color: Ocean Blue — #0EA5E9
Background color: Soft Light — #F8FAFC
Dark mode background: Deep Navy — #0F172A
Accent color: Pomegranate Red — #D72638

4. Typography feel

Modern and readable.
Friendly but not playful.

5. Layout direction

Use a clean homepage with a strong hero section, a bento-style feature area, and a preview of homestay cards.

The directory page should use a responsive card grid with spacious cards, large images, price, location, and a clear “View Details” button.

The details page should feel like a simple Airbnb-style listing page with images on top, key info below, facilities in clean sections, available dates as simple text, and a sticky or highly visible WhatsApp button.

6. Component style

Buttons should feel slightly premium with rounded corners, soft shadows, subtle hover movement, and clear depth.

Cards should be rounded, image-led, spacious, and slightly elevated with gentle shadows.

Sections should use soft backgrounds, bento blocks, good spacing, and small micro-interactions like hover lift, fade-in, and button glow.

7. Images or icons

Use warm, real-looking homestay images: kampung houses, modern rooms, family spaces, outdoor areas, beaches, gardens, and holiday-style interiors.

Use simple line icons for facilities like WiFi, parking, aircond, bedrooms, kitchen, and WhatsApp.

Avoid generic hotel stock photos, overly corporate visuals, crowded booking-app layouts, and anything that feels too much like Booking.com.
