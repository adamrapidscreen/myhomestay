Project Brief
Goal

Build a simple homestay directory website that helps visitors browse homestays, view key details, and contact the owner directly through WhatsApp.

Audience

This website is for people looking for homestays and homestay owners who want a simple way to advertise their place online.

Stack

Next.js + Tailwind CSS + Vercel

This stack is enough for a beginner because the website can be built as a simple static site, styled easily with Tailwind CSS, and deployed quickly on Vercel without needing a database or backend.

Pages
Homepage
Introduces the website and has a button to browse homestays.
Directory Page
Shows all listed homestays in simple cards.
Homestay Details Page
Shows full details for one homestay, including pricing, facilities, available dates, and WhatsApp contact button.
Features
Homestay directory cards
Display homestay name, image, location, price, and short facilities preview.
Homestay details page
Show full information for each homestay, including pricing, facilities, photos, and available dates.
WhatsApp contact button
Let visitors message the owner directly with a prefilled message.
Non-goals

We are intentionally not building these yet:

Login system
Owner dashboard
Online booking system
Payment system
Real-time availability calendar
Reviews or ratings
Search and filter system
Admin panel
Build order
Set up the Next.js project with Tailwind CSS.
Create a simple homepage.
Add sample homestay data manually in a data file.
Build the directory page using homestay cards.
Build the homestay details page.
Add the WhatsApp button with a prefilled message.
Make the website responsive for mobile and desktop.
Deploy the website on Vercel.
Success check

The first version is good enough when a visitor can open the website, browse homestays, view one homestay’s details, understand the price/facilities/availability, and contact the owner through WhatsApp without confusion.
