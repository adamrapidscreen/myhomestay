# Beta 0.17 - Launch Bloom Pack

Date: 2026-06-01
Release marker: Beta 0.17 (`package.json` 0.17.0)
Workflow: KD Plan & Build pilot

## Summary

The Launch Bloom Pack adds a launch-quality trust, sharing, and conversion layer
around owner-created listings, without turning the MVP into a marketplace. No
payments, bookings, traveller accounts, reviews, ratings, subscriptions, or
public verification claims are introduced.

Five features, delivered as Work Cards LB-1 through LB-7:

1. Owner Trust Checklist
2. Share Kit
3. Traveller Confidence Strip
4. WhatsApp Inquiry Kit
5. Local Mini-Guide

## What changed

### Data foundation (LB-1)

- New migration `supabase/migrations/0012_launch_bloom_pack.sql`.
- Tables: `listing_guide_items` (owner-managed nearby notes), `listing_reports`
  (public abuse/quality intake, admin-readable only).
- Enums: `guide_item_category`, `listing_report_reason`, `listing_report_status`.
- RLS: owner-scoped guide CRUD via listing ownership; admin-only report read/update.
- Definer RPCs: `get_published_listing_guide_items` (column-scoped public read,
  published-only) and `submit_listing_report` (published-only intake; no public
  table INSERT policy). Mirrors the Chapter 4 public-read decision.

### Owner Trust Checklist (LB-2)

- Computed 9-check launch readiness checklist (`src/lib/launch-checks.ts`).
- New owner launch hub at `/dashboard/listings/[id]/launch`.
- Dashboard row "Launch kit" link and edit-page launch summary banner.
- Evidence-based language only; no verified/certified/approved claims.

### Share Kit (LB-3)

- Public link copy, native share, WhatsApp share, and caption starters.
- Server-side QR route `/dashboard/listings/[id]/qr` (owner + published only),
  using pinned `qrcode@1.5.4`. QR encodes only the public listing URL and is kept
  out of the client bundle.

### Local Mini-Guide (LB-4)

- Owner guide-item manager (add/delete) on the launch route.
- Public grouped "Nearby and local notes" section on the listing page, read via
  the column-scoped RPC. Hidden entirely when empty.

### Traveller Confidence Strip and Report Listing (LB-5)

- Factual confidence strip on the public listing detail page.
- Quiet "Report this listing" form posting to `/api/listing-reports`.
- Admin read-only reports section on `/admin`.

### WhatsApp Inquiry Kit (LB-6)

- Inquiry composer replaces the plain CTA card: optional dates, guest count, and
  question build a prefilled WhatsApp message locally. Nothing is stored; the
  WhatsApp click metric still fires.
- Owner copy-paste quick replies helper.
